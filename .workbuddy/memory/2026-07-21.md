- 关键发现：设计系统全部颜色走 `app/src/index.css` 的 token（`--color-*`/`--surface-*`），组件禁止硬编码颜色/像素（`shared/ui/CLAUDE.md` 硬规则）。`index.css` 是唯一真相源。
- 方案：**零改动换皮**——产出独立覆盖文件 `supernote-dark-theme/theme-dark.css`（在 index.css 之后 import，给 `<html>` 加 `data-theme="dark"` 生效），重映射 token + 重写暗态阴影 + 对"把 --color-ink 当深色漆"的类（primary 按钮/ink 卡/弹窗遮罩/激活树节点）做定向 CSS 覆盖。**未编辑 index.css、未改任何组件/功能**。
- 配色：暖调石墨 warm-charcoal（putty→#161513, bone→#1d1b17, ink→暖白 #ECE8E0, paper→#0F0E0B），承接浅色纸感品牌。
- 交付：`supernote-dark-theme/` 下 `theme-dark.css`（生产用）、`dark-theme-preview.html`（可双击打开、右上角一键切换 当前浅色↔新暗色 对照）、`README.md`（方案+激活+取舍）。
